Microsoft Virtual Academy introduices the Learning Paths

Microsoft added some Learning Path on his online training platform, Microsoft Virtual Academy.

The Learning Paths are some lessons group together relative to a theme, or a technology. The objective is to help you build some valuable skills with lessons, demos and assestments.

You earn a badge when you complete a session, a kind of "light" certification you can share on social medias.
