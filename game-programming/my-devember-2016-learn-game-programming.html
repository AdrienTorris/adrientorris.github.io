My #devember : learn game programming

This is a long time desire I grow back for a while : understand how the games are made. Each time I play FIFA, this thought doesn't leave me : how it is possible to reproduce with that precision so much physics effects ? How are managed the different players behaviors ? How is built the AI for the soccer who is a very complex sport ?

So this year, I decided to not grow back again and to understand what happens when you launch a video game. Happily, I found an amazing tutorial I write on later who teaches who to develop a game from scratch in C so let's go. My devember challenge this year : learn C game programming. Finally.
